# 🏦 Bank Customer Transaction Analysis Dashboard
## 📌 Project Overview

This project presents an interactive Power BI dashboard developed to analyze customer demographics, transaction behavior, account balances, and location-based insights for a banking dataset.

## 🎯 Business Objective

The goal of this project is to help banks understand customer transaction patterns, identify high-value customers, and support data-driven decision-making.

## 🛠 Tools Used

* Power BI
* DAX
* Power Query
* Excel/CSV

## 📊 Dashboard Features

### Executive Dashboard

* Total Customers
* Total Revenue
* Average Transaction Amount
* Average Account Balance
* Highest Transaction Value
* Gender Distribution
* Age Group Analysis
* Location Analysis
* Transaction Trends

### Advanced Analysis

* Scatter Plot Analysis
* Funnel Analysis
* Customer Segmentation Matrix

## 🔍 Key Insights

* Identified top-performing customers based on transaction value.
* Analyzed customer distribution across locations and gender.
* Examined transaction trends over time.
* Compared account balances with transaction activity.

## 🚀 Skills Demonstrated

Power BI, Data Visualization, DAX, Power Query, Data Cleaning, Dashboard Design, Business Analytics.
